# finance-analytics-dashboard-powerbi

Project Overview

This project presents an interactive Finance Analytics Dashboard built in Power BI to analyze financial transactions, customer behavior, operational fees, taxes, and business performance across multiple dimensions.

The dashboard enables stakeholders to monitor key financial KPIs, identify growth opportunities, evaluate customer segments, and make data-driven business decisions.

Business Problem

Financial organizations often struggle to:

Monitor overall transaction growth
Analyze monthly transaction trends
Track successful vs failed transactions
Identify high-value customer segments
Compare state-wise financial performance
Evaluate transaction-type profitability
Understand customer demographics
Measure Year-over-Year (YoY) performance

This dashboard provides a centralized analytical solution to address these challenges.
